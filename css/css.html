CSS (Cascading Style Sheet) HTML 문서의 요소(태그)의 모양이나 색깔, 디자인을 꾸밀때 사용한다 페이지의 레이아웃을 구성 웹페이지의 내용을 HTML로만 구성하고 디자인은 CSS로하자 => 웹 표준 스타일시트의 종류 1. 내부 스타일 시트 : html 파일 안에서 를 이용해서 지정 2. 외부 스타일 시트 : 문서 외부에 따로 css 파일을 이용해서 지정 3. 인라인 스타일 : 태그(요소) 안에서 style 직접 작성 속성값을 지정할때 스타일이 지정되는 순서(우선순위) 외부스타일 << 내부스타일 << 인라인스타일 스타일이 3개가 겹치면 우선순위가 가장 높은 스타일을 적용 Cascading : 물처럼 흐르는, 위에서 아래서 흐르는 h1 { color : red ;} 선택자 스타일속성 스타일값 css 블록
