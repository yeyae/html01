<> 첫줄의 DOCTYPE 태그는 웹브라우저가 현재 웹페이지가 HTML 문서임을 인식하게 한다. HTML 문서는 반드시 첫줄에 DOCTYPE태그가 있어야 한다. 두번째줄의 html 태그는 모든 html 페이지의 루트 요소이다. html 태그에는 lang 속성을 입력할수 있다. lang 속성은 실제로 웹 브라우저가 동작하는데 영향을 끼치지 않음. 대신에 구글같은 검색엔진이 웹 페이지를 탐색할때 해당 페이지가 어떤 언어로 작성되어있는지 판단하기 쉽게 만들어준다. head 태그 body태그에 필요한 외부 자원들을 제공하는 공간 css (스타일시트), javascript 태그의 종류 meta : 웹페이지에 추가 정보를 전달 title : 웹페이지의 제목 script : 웹페이지에 스크립트를 추가한다. (javascript) link : 웹페이지에 다른 파일을 추가한다. (이미지, sript,css) style : 웹페이지에 스타일시트를 추가합니다. (css) body 태그
