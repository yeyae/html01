HTML (HyperText Markup Language) 웹브라우저 (크롬, 엣지, 익스플로러) 에서 보이는 화면을 그리기 위한 언어 Tag <태그이름> 태그를 이용해서 문서를 구분하기 때문에 markup 언어라고 한다 (이미지, 버튼지, 글자, 제목인지) 요소 element 태그와 요소는 본질적으로 같은 것을 가리킨다.(구분하지 않는다.) 문맥에 따라서 더 적합한 단어를 사용할때가 있음. 속성 attribute 태그에 추가 정보를 부여할때 사용하는 기능

HELLO HTML

title => 속성이름 "header" => 속성값 src => 속성이름 "image.jpg" => 속성값
